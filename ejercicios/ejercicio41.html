Glosario de Etiquetas de HTML

html: Define el inicio del documento HTML. Es la etiqueta raíz que contiene todo el contenido del sitio web.

head: Contiene metadatos del documento, como el título, enlaces a archivos de estilo, scripts, etc. No es visible en la página.

title: Define el título del documento, que se muestra en la pestaña del navegador.

body: Contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, etc.

h1 a h6: Etiquetas de encabezados que van desde el nivel 1 (más importante) hasta el nivel 6 (menos importante).

p: Define un parrafo

a: Crea un enlace a otra pagina o recuerso mediante una url

img: Inserta una imagen en la página. Usa el atributo src para definir la fuente de la imagen.

ul: Crea una lista desordenada(con viñetas).

div: Un contenedor genérico utilizado para agrupar contenido o aplicar estilos y diseño a bloques de contenido mediante CSS
